h1
This is a heading one tag. It tags text and declares it as a heading one.
h2
This is a heading two tag. It tags text and declares it as a heading two.
h3
This is a heading three tag. it tags text and declares it as a heading three
p
this is a paragraph tag. It tags the text and declares text to be a paragraph
ul
This is an unsorted list. It tells the browser that a list will follow.
ol
This is an Organized list. It tells the browser that an organized "numbered" list will follow
li
This is a list item. it tags the text as list item nested within an organized or unorganized list. It should only be used within an organized or unorganized list.
